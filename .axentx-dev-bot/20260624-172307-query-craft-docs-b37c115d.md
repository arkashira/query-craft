# Query‑Craft

`query-craft` is a new command‑line tool that lets you build, test, and run complex query pipelines with minimal boilerplate. It supports:

- **Declarative pipeline definition** via a simple YAML/JSON file.
- **Automatic dependency resolution** between query stages.
- **Built‑in performance profiling** using k6 2.0.
- **Extensible plug‑in system** for custom query adapters.

See the [docs](docs/query-craft.md) for installation, configuration, and usage examples.
